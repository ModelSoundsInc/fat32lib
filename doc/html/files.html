fat32lib
fat.hThis is the header file for the FAT driver. It allows low level access to the file system. The functions defined in smlib\sm.h allow access to all the functionality provided by this header, most users should use those instead. The only all users need to call on this module is fat_get_filesystem_interface which is defined in filesystem_interface.h and allows smlib (the volume manager) to interface with fat32lib (the file system driver.)
